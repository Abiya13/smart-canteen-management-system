# Smart Canteen Management System

A web-based Smart Canteen Management System developed using Django. The system helps users pre-book food, place orders online, make payments, and collect food using generated tokens.

## Features

- User Registration and Login
- Role-Based Authentication
- Food Menu Management
- Add to Cart Functionality
- Buy Now Functionality
- Pre-Booking System
- Online Payment Integration
- Token Generation After Payment
- Order Management
- Admin Dashboard
- Staff Dashboard
- Order Tracking

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap

## Installation

```bash
git clone https://github.com/Abiya13/smart-canteen-management-system.git
cd smart-canteen-management-system
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
